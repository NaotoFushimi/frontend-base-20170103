# frontend-base-20170103

Boilerplate for modern frontend environment (typescript + babel + modern CSS)

## Core Modules
* babel
* webpack
* typescript



## How to use

```
git clone https://github.com/NaotoFushimi/frontend-base-20170103.git
```

init
```
rm -rf .git
yarn install
```


watch & build
```
yarn run watch

```
